111/*
112 * === EASY DIFFICULTY WRITER API ===
113 *
114 * Use a static helper function to create a VTF in one function call - VTF::create
115 *
116 * === MEDIUM DIFFICULTY WRITER API ===
117 *
118 * Constructing a VTF instance from existing VTF data will let you modify that data.
119 *
120 * This class has methods that should be called in a particular order
121 * when creating a VTF from scratch, or your output VTF may look incorrect:
122 *
123 * 0. Construct an empty VTF instance
124 * 1. Set the version (7.4 is the default) - VTF::setVersion
125 * 2. Set FLAG_SRGB (optional, read for image resizing) - VTF::setFlags
126 * 3. Set the image resize methods (optional) - VTF::setImageResizeMethods
127 * 4. Set the base image (mip 0, frame 0, face 0, slice 0) - VTF::setImage
128 * 5. Compute mips (optional) - VTF::computeMips
129 * 6. Set the output format (optional) - VTF::setFormat
130 *
131 * After these methods are called, the other writer methods in the class should work as expected.
132 */
